# She Software Solutions — AI Chatbot

An intelligent AI-powered chatbot for She Software Solutions website.

## 🔗 Live Demo
https://she-software-chatbot.vercel.app

## ✨ Features
- AI-powered responses using Claude API
- Company information assistance
- Internship guidance
- Lead collection system
- Admin panel with chat logs, FAQ manager, leads view
- Firebase Firestore database integration
- Fully responsive UI

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Database: Firebase Firestore
- AI: Claude (Anthropic API)
- Deployment: Vercel (frontend) + Render (backend)

## 🚀 Setup Instructions

### 1. Clone the repo
git clone https://github.com/Jaya12563/she-software-chatbot.git

### 2. Backend Setup
cd backend
npm install
Create .env file with:
ANTHROPIC_API_KEY=your_key
PORT=8080
node server.js

### 3. Frontend Setup
cd frontend
npm install
npm start
